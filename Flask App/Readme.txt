E-commerce Dashboard

This Flask application represents a simple dashboard for analyzing data from the e-commerce platform. The dashboard provides various visualizations and insights into order status, product categories, customer analysis, seller analysis, customer satisfaction, payment methods, and more.

Clone the repository:
https://github.com/muhamedabdlnapy/E-commerce-Analysis

Navigate to the project directory:
cd FlaskAPP


Create a virtual environment:
python3 -m venv venv
source venv/bin/activate

Install the required dependencies:
pip install -r requirements.txt
